Site Asumarte

O site Asumarte, foi um projeto realizado em ambiente acadêmico, que tinha o objetivo de fazer um site para essa marca, com fim de experiência para os alunos.
Inicialmente, era para ser feito um site com uma paleta de cores escolhida pelos donos da marca, Ter uma forma de contato, login, portifolio, uma loja e uma aba para informarções(sobre). Eles permitram que os alunos pudessem desenvolver o resto do site, de acodo com a criatividade, respeitando os limites que impuseram em determinadas direções. Obs: Esse não é o site é apenas um projeto acadêmico. Site oficial
